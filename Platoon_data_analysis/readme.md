## 一、 分层设计思想：
- 公共的方法目录：common
- 测试数据目录：data
- lcm结构体：lcmtypes
- 测试报告存放目录：reports
- 配置文件存放目录：conf
- 日志文件存放的目录：logs
- 项目的启动文件：run_data.py
- 监听类文件：listener


## 二、数据驱动思想
- 读取指定的lcm数据，读取完成以后选择想要读取的数据


## 三、程序运行流程思路：
- 1、读取data目录下的lcmlog文件
- 2、通过数字选择需要读取的数据
- 3、读取选中的数据
- 4、生成图片存放到logs文件内
